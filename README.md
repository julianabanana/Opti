# Optimización - Universidad Nacional de Colombia

Este repositorio contiene el desarrollo académico, implementaciones algorítmicas y soluciones a problemas de optimización lineal y no lineal correspondientes al curso de la UNAL.

🌐 **Recurso Oficial del Curso:** [Optimización ](https://optimizacionunal.netlify.app/)

---

## 🎯 Objetivo del Repositorio
Documentar de manera estructurada el proceso de aprendizaje y resolución de problemas de optimización, abarcando desde el modelado matemático hasta la implementación computacional.

### Herramientas Utilizadas:
* **Latex** 
* **Python**
* **Matlab** 

---

## 📂 Estructura del Proyecto

A petición del profesor, las entregas se realizan por ramas del repositorio.
